## i18n (двуязычность RU/EN)

- **Полная система переключения языка** на все 34 экрана
- ~160 текстовых ключей с `data-i18n` атрибутами
- 6 placeholder'ов с `data-i18n-placeholder`
- Typewriter тексты, CoachMark тексты, динамические прогресс-лейблы — всё переключается
- **Переключатель** — pill toggle RU/EN на экране 2 (Alex Appears) и в Профиле (экран 12)
- **localStorage** — язык сохраняется между сессиями (`sw-lang`)
- **Определение языка:** в проде — `navigator.language`, fallback на RU

### Как реализовать в проде
- Каждый элемент с `data-i18n="key"` — заменить на серверную/клиентскую i18n (react-i18next или аналог)
- Словарь текстов: `i18n` объект в JS содержит все ключи (RU + EN), можно экспортировать в JSON

## Баг-фиксы

1. **CoachMarks z-index leak** — после завершения туториала z-index не сбрасывался на всех элементах, только на последнем. Карта мира перекрывала profile header. Починено: `skipCoachMarks()` сбрасывает все элементы
2. **Profile header clickability** — никнейм кидал на карту мира вместо профиля. Починено: `position: relative; z-index: 5` на header
3. **Auto-transition Splash** — сломался при добавлении geo-block. Восстановлен: триггер при переходе на экран 1 (Splash), не при загрузке
4. **q2Screens номера** — не обновились при перенумерации. Починено: `wallet: 27, verify: 28, card: 29, deposit: 30, txn: 31`
5. **Хардкоженный русский в inline JS** — ошибки валидации никнейма, "Заполни все поля", "Готово ✓" — теперь language-aware
